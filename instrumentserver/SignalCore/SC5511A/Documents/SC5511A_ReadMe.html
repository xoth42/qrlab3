Software Packages Needed

The Web-based installed package does not include the National Instruments LabVIEW runtime Engine that is required to run the device software front panel. The LabVIEW RunTime Engine version 2010 can be download from National Instruments's website here (may need to create an account and login):

http://www.ni.com/download/labview-run-time-engine-2010-sp1/2292/en/

You will also need LabVIEW version 2010 or later to run the LabVIEW API funtions. The Software Front Panel GUI is generated from LabVIEW and its source code is in the \example directory of the device LabVIEW directory under the installation path, for example  C:\ProgramFiles\SignalCore\SC5511A\api\labview\SC5511A\


Driver Installation

When your device is properly connected into your computer, and the computer detects the device, it will show up in the device manager screen. If this is the first time a SC5511A is installed on the computer, it will appear as a USB Device under Other Devices. Perform the following steps to install the driver.

  1. You may run device manager by typing in devmgmt.msc into the search text bar as you click the [Start] button.
  2. Find the USB Device and right click, and select Update Driver Software.
  3. Choose to Browse .. . for the driver, and on the following screen select Let me pick from .. .
  4. Choose the button [Have Disk .. .] and browse to the installation path of the device. A typical path would be C:\Program Files\SignalCore\SC5511A\driver\x64 for a 64bit system.
  5. The computer should identify the device as SC5511A RF Source. Click OK to install the device.
  6. Upon successful installation the device will now appear in the device manager as SC5511A RF Source under the SignalCore USB Devices
