/* Instructions to use the sc5506a shared library */

The driver library and code is written based on libusb-1.0, please visit http://libusbx.org for the latest version. 
The driver provided by SignalCore Inc. has been tested on libusb-1.0.8 or later on a Debian distribution of the Linux opearating system. Please contact SignalCore for support questions.

On your console terminal type "make all" to make the libraries
Type "make install" to install
Make sure you have root privileges.
